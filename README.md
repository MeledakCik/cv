# 🛡️ SENTINEL - Security Focused Portfolio

A professional digital identity and security monitoring dashboard built for the **GDG Bandung #BelajarSambilBeramal Season 6 Challenge**. 

This isn't just a portfolio; it's a live demonstration of web hardening, real-time analytics, and modern full-stack architecture.

## 🚀 Technical Excellence
- **Framework:** Next.js 16 (App Router) with Turbopack for ultra-fast builds.
- **Styling:** Tailwind CSS with a "Cyber Lab" dark theme aesthetics.
- **Analytics:** Integrated Recharts for real-time skill mapping and traffic auditing.
- **Security:** Hardened headers, rate limiting, and automated system detection.

## 🛠️ Core Features
- **Sentinel Dashboard:** Real-time monitoring of Node IP, system load, and request traffic.
- **Skill Radar:** Visual representation of technical expertise in Cyber Security, Python, and Web Dev.
- **Security Audit Logs:** Live-simulated terminal logs showing system activity.
- **Automated Device Detection:** Identification of OS and Terminal access points.
